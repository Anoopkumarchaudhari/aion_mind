==============================================================================
 AION MIND — Requirements
==============================================================================
 This is a Next.js (TypeScript) application. JavaScript projects do not use a
 Python-style requirements.txt; the authoritative dependency manifest is
 package.json (installed via `npm install`). This file is a human-readable
 summary of everything needed to run the app.
==============================================================================

------------------------------------------------------------------------------
 SYSTEM PREREQUISITES
------------------------------------------------------------------------------
Node.js        >= 18.18.0   (Next.js 15 requirement; Node 20 LTS recommended)
npm            >= 9         (ships with Node)
PostgreSQL     >= 14        (database; see DATABASE_URL below)
Prisma         (schema in /prisma — run `npx prisma generate` / `migrate`)

------------------------------------------------------------------------------
 INSTALL & RUN
------------------------------------------------------------------------------
npm install              # install all dependencies from package.json
npx prisma generate      # generate Prisma client
npm run dev              # start dev server (http://localhost:3000)
npm run build            # production build
npm start                # run production build
npm run typecheck        # tsc --noEmit
